A full program of the UAV (Unmanned Aerial Vehicle), specifically the hexarotor for the team SIRI of Soegijapranata Catholic University in Indonesian Aerial Robotic Contest (KRTI - Kontes Robot Terbang Indonesia) 2024.

- Software for autonomous attitudes is based on Python, and for computer vision is based on the PyTorch implementation.
- Hardware based on the PixHawk-based microcontroller as the slave controller for the attitudes, and Raspberry Pi 4 as the main controller and computer vision processing power.
